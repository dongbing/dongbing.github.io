PVE+LEDE

准备工作

  1. 去Proxmox官网下载镜像。
  2. 下载lede x64安装镜像。
  3. 下载img2kvm工具，用来把lede镜像转为kvm磁盘。
    img2kvm

安装PVE

将pve安装镜像写入U盘，使用uefi模式安装系统。

安装LEDE系统

在PVE中新建VM，不用建立硬盘。

配置网络

在PVE中 Create -> Linux Bridge 添加所有的网卡，管理接口作为LAN口，IP地址设置为：

作为WAN口的网卡IP地址不用设置。

在虚拟机中添加网卡，Model均设置为VirtIO。

开启LEDE虚拟机，在网络接口中将LAN口设置为192.168.2.1即可正常上网。
