安装前准备

Hexo安装前需要确保有: Node.js Git
去Node.js官网下载源码。

可以使用node -v检查是否安装成功。

开始安装

Node.js安装完成后即可安装Hexo

Hexo初始化

Hexo安装完成后初始化本地文件

完成后进入修改_config.yml文件，具体配置可以去Hexo官网查询。
Hexo主题可以从官网下载，解压后复制到themes文件夹，配置中修改即可。

使用Hexo新建博客文章

  1. 新建文章

    新建博客文章，相应的markdown文件在source/_posts文件夹中，修改即可。

  2. 生成网站

    生成静态网站，整站在public文件夹下面。

  3. 网站预览

    可以预览生成的静态网站。
