Windows远程桌面提示您的凭据不工作

服务器端

进入组策略编辑器。
计算机配置→Window设置→安全设置→安全选项→网络访问：本地账户的共享和安全模型→经典-对本地用户进行身份验证，不改变起本来身份

客户端

进入组策略编辑器。
计算机配置→管理模版→系统→凭据分配→允许分配保存的凭据用于仅NTLM服务器身份验证→启用→输入“TERMSRV/*”
